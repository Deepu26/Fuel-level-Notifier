
In Android application the following steps are done.

1.It will request the cloud (think speak) to get the data from channel .
2.By initializing the read key, which was given by particular channel , we will be able to get the data.
3.After getting the data it is converted to percentage .
4.The user will be able to get an alert message when the fuel level goes below the threshold value (20 %).
5.The users can view the diesel level  at anytime. 
6.For security purpose app has an login page (default credentials are given by developer).
